# 🚗 Car Repair Invoice System - Professional Invoice Generator

A modern, multi-user web application for car repair workshops to create professional invoices, track earnings, and manage customer records with WhatsApp integration.

## ✨ Features

### 📋 Core Features
- **Instant Invoice Generation**: Create professional PDF invoices in seconds
- **Multi-User Support**: Each mechanic gets their own private workspace
- **Real-time Statistics**: Track daily earnings, invoices, and sales
- **Labor vs Parts Calculation**: Accurate earnings tracking (labor only)

### 📱 Smart Integrations
- **WhatsApp Integration**: Send invoices directly to customers
- **Auto PDF Generation**: Professional formatted invoices with workshop details
- **Data Persistence**: All data saved securely with user isolation

### 🛠️ Workshop Management
- **Custom Profiles**: Set your workshop name, phone, address, logo
- **Profile Editing**: Update details anytime in sidebar
- **Multi-Mechanic Ready**: Perfect for team workshops

### 📊 Business Insights
- **Daily Earnings Dashboard**: Real-time labor earnings tracking
- **Invoice History**: Access all past invoices
- **Performance Analytics**: Average invoice value, items sold, etc.
